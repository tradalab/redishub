# Roadmap

Our vision for RedisHub is to build the ultimate command center for the Redis ecosystem. Here is our strategic development map.

## Recently Released
- [x] **Native Redis Cluster Support**: Automatic node discovery and slot routing.
- [x] **Sentinel Integration**: High-availability master discovery.
- [x] **Advanced TLS/mTLS**: Enhanced security with mutual certificate authentication.
- [x] **SSH Tunneling v2**: Improved stability and authentication methods.

## (In Progress)
- [ ] **Redis Hub Web**: A lightweight, cloud-ready web interface for browser-based access.
- [ ] **Advanced Filtering**: Support for complex Regex and multi-pattern search in the key browser.
- [ ] **Performance Engine**: Optimized scanning for databases exceeding 10M+ keys.

## (Planned)
- [ ] **Monitoring Dashboard**: Real-time visualization of memory usage, hits/misses, and command throughput.
- [ ] **Visual Query Builder**: Create complex Redis queries via an intuitive drag-and-drop interface.
- [ ] **Lua Scripting Suite**: Professional editor with syntax highlighting, debugging, and script library management.

## (Future Vision)
- [ ] **Collaboration Suite**: Share connection profiles and analysis snippets securely with your team.
- [ ] **Data Transformation Engine**: Built-in tools for seamless data migration and format conversion (JSON, CSV, Protobuf).
- [ ] **Plugin Ecosystem**: Open API for community-driven extensions and custom themes.

---
*Note: This roadmap is driven by the community. Features are prioritized based on user feedback and technical impact.*
